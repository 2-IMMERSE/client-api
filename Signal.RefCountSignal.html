registerReferenceSignal(signal, listenerTrackeropt)

Calls Signal.RefCountSignal#registerReference when the value of the signal is highy/truthy, and Signal.RefCountSignal#unregisterReference when the value of the signal is low/falsey, as necessary.
This can be used to follow the state of another reference count signal, for example.

Note that remove the event listener using the listenerTracker does not unregister the reference.
